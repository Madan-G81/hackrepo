# FinTrack – Expense Tracker Web App

A modern, production-ready expense tracker built with **HTML, CSS, and JavaScript**. It features a premium fintech-style UI, glassmorphism, dark/light mode, responsive charts, and complete localStorage persistence.

---

## 🔗 Live / Repo
- **GitHub Repo:** https://github.com/Madan-G81/hackrepo

---

## ✨ Features
- Dashboard with **Total Balance, Income, Expense** and animated counters
- Add/Edit/Delete transactions with validation
- Search and filter by type, category, and date
- Default categories included
- Pie + Bar charts (responsive)
- Dark/Light mode with persistence
- Budget limit warning & reminder UI
- Export CSV and download report
- Toast notifications
- Mobile-first responsive layout

## 🚀 How to Run Locally
1. Clone or download this repository.
2. Open **index.html** in your browser.

No backend required.

---

## ☁️ Deploy to GitHub Pages
1. Go to **Settings → Pages** in your GitHub repo.
2. Under **Source**, select:
   - Branch: `main`
   - Folder: `/ (root)`
3. Click **Save**.
4. Your app will be live at:
   ```
   https://madan-g81.github.io/hackrepo/
   ```
